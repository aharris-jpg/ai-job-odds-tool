How We Built This Tool What comes from Anthropic, and what Vegas Insider added. +

This tool starts with occupation-level AI exposure data from Anthropic, which estimates how much of each job overlaps with tasks AI can already perform or assist with.

Vegas Insider then transformed that source data into this odds-based ranking.

  1. Anthropic provides the source exposure score for each occupation.
  2. Vegas Insider uses that score as the base signal of AI task overlap.
  3. Vegas Insider converts it into a probability-style metric for easier comparison across jobs.
  4. Vegas Insider ranks occupations from highest to lowest using that modeled odds figure.
This tool measures the odds AI replaces major parts of a job, not the odds a job disappears completely.
